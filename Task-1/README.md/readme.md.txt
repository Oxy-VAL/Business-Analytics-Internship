# Task 1 - Market Basket Analysis

## Objective

Identify products that are frequently purchased together using Market Basket Analysis and generate business recommendations to improve cross-selling opportunities.

## Dataset

- Groceries Dataset
- Transaction-based retail purchase records

## Techniques Used

- Data Preprocessing
- One-Hot Encoding
- Apriori Algorithm
- Association Rule Mining

## RetailIQ Enhancement

### Product Recommendation Engine

The RetailIQ module transforms association rules into actionable business recommendations by:

- Identifying cross-selling opportunities
- Suggesting shelf placement improvements
- Supporting promotional bundle creation

## Business Value

- Increase Average Basket Value
- Improve Cross-selling
- Optimize Product Placement
- Support Retail Decision Making
